# Samy Test Front

This project is a Single Page Application (SPA) that displays images in an infinite scrolling mode

## Run Mock server

In the directory `/mocks` you can run:

1. Install node module

```bash
npm install
```

2. Run Mock server

```bash
npm run mocks
```

## Run React App

In the project directory, you can run:

1. Install node module

```bash
npm install
```

2. Run app

```bash
npm start
```
